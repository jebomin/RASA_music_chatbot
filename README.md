## ▶️ 실행 방법
### 1. 필요한 api 설치
아나콘다 프롬포트에서 가상환경 생성 후 설치</br>
```bash
pip install rasa==3.6.4
```

### 2. 실행
visual studio code에서 해당 파일을 다운로드 받은 후 터미널에 다음 명령어 입력</br>
```bash
ran run actions --port 6000
rasa train
rasa shell
```
